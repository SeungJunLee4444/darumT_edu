## 소프트웨어 구조

# 소프트웨어는 os와 application으로 나뉜다
* cpu위에 os, os 위에 프로그램(application)이 실행된다

1) os / 운영체제
- 개념 : 사용자가 하드웨어에 쉽게 접근할 수 있도록 하는 소프트웨어
- 종류
	(1) 유닉스 : 서버운영에 필수적인 cli에 특화된 체제
	- 윈도우 외 모든 운영체제는 유닉스에서 비롯되어, 유닉스의 명령어가 공통적으로 사용된다
	(2) 리눅스 : 다중사용자, 다중작업, 다중스레드 가능
	(3) 윈도우 : 기존의 cli 방식 대신 마우스로 컴퓨터를 다루기 위해 탄생한 os

2) application / 운영 소프트웨어 
(1) 워드 프로세서
(2) 스프레드 시트
(3) 컴파일러
(4) 웹 브라우저
