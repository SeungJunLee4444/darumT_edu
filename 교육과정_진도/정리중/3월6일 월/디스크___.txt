## 디스크

1. 하드디스크(hdd)
- 특징 : 많은 데이터를 저렴하게 저장이 가능
- 단점 : 움직이는 부품, 물리적인 충격으로 인한 손상가능성이 높음

2. ssd 
- 개념 : hhd를 개선한 디스크, 움직이는 부품이 없어 hdd보다 더 빨리 데이터를 읽을 수 있고, 엑세스 가능
- 특징 : 파일에 빨리 엑세스 하기에 적합, 수명이 김
- 단점 : raid가 안됨


4. 디스크 파티션
- 뜻 : 하드 드라이브를 나누는 것을 말한다(ex c와 d로 나누기)

5. 파일시스템
- 정의 : 파일을 사용자가 쉽게 접근할 수 있도록 운영체제가 시스템의 디스크에 보관하는 것을 말한다
파일 시스템이 있기에 컴퓨터에 파일을 저장하고 보관할 수 있다
- 종류
1) 리눅스의 파일시스템 : 파일시스템은 바꾸는 순간 포멧되기 떄문에 백업을 해야한다
(1) xfs 	: 대용량 파일시스템(16테라바이트 이상), redhat7 출시 이후 기본파일시스템으로 채택
(2) ext4	: 대용량 파일 지원 및 빠른 저장공간 할당, 최대 16테라바이트까지 지원

* os안에서 지원하는 파일시스템 내에서는 혼용할 수 있다
하지만 대부분은 하나를 선택해서 사용한다
