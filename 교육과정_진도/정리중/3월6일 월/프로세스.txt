## 프로세스

- 개념 : 서버의 cpu에서 실행되는 프로그램을 말한다
- 구성	
(1) pid	: 프로세스의 고유 id
(2) ppid	: 부모 프로세스의 id
	- 같은 부모 프로세스들 최상단 pid가 부모 프로세스
	- 해당 부모 프로세스를 종료하면 상속되는 모든 프로세스가 종료된다 *
	- 부모 프로세스를 죽여도 살아있는 좀비 프로세스가 발생할 수 있음 * 
	- 모든 프로세스들의 최상단에 위치한 프로세스들은 '조상프로레스'로 부팅시 실행되며 systemd라는 이름이 붙는다 *
(3) uid	: 사용자 id
(4) c	: cpu 점유율
(5) stime	: 시작시간
(6) tty	: 접속 도구
(7) time	: 명령어 실행시간
(8) cmd	: 명령어
