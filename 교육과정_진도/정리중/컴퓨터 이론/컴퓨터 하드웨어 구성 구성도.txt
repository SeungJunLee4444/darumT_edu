## 컴퓨터 구성도

1. 컴퓨터 구조
1) cpu / 중앙처리장치		
- 개념 : 메모리에서 보내진 연산처리가 목적인 장비
- 제조사 
	(1) 인텔 : 한가지 프로그램을 돌리는 것에 최적화
	(2) amd : 한번에 여러 작업을 처리하는데 최적화

* cpu 캐시 : cpu에 위치하여 메모리로 사용되도록 구성된 하드웨어 캐시
- 대부분의 메모리 접근은 특정한 위치의 근방에서 자주 일어나는 경향이 있음
- 이 데이터를 캐시 메모리에 복사해두면 메모리 접근 시간을 아낄 수 있음


2) 메모리(mem) / 주기억장치
- 개념 : 데이터는 보조기억장치에 있으며, cpu에서 바로 가져올 수는 없는데,
연산처리를 위한 데이터를 보조에서 가져오기 위한 장치
- 특징 : 메모리는 현재 실행되는 프로그램(프로세스)을 저장
ex) 브라우저

3) 보조기억장치
- 개념 : 전원이 꺼져도 데이터를 저장하는 장치(비휘발성)
- 종류 : 하드디스크, ssd, usb, dvd, cd
- 특징 : 보조기억장치는 보관할 프로그램을 저장
ex) 사진, 동영상

4) 입출력장치 : 마이크, 스피커, 키보드 등 컴퓨터 외부와 내부와 정보 교환을 위한것
