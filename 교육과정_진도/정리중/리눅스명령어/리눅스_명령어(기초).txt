##  리눅스 명령어(기초)

# 기본
- clear	: 화면정리
- exit	: 로그아웃
- who, whoami	: 현재 접속한 계정 확인
- alias	: 단축키 확인

# 프롬프트 구조	: 계정@호스트명 현재 작업중인 디렉터리 위치 # / $


# 다른계정으로 접속 
	- su
	(1) su -		: root 계정으로 접속
	(2) su - 계정명	: 다른 사용자 계정으로 접속
	* -를 추가해야 사용하던 환경변수(쉘)을 그대로 사용가능

# 리눅스 접속 유지
	- top : 로그인 이후 일정시간이 지나면 자동 로그아웃되는걸 방지

# 다른 디렉터리로 이동
	- cd
	(1) cd ~ 		: 홈 디렉터리로 이동
	(2) cd /폴더	: 특정 폴더로 이동

# 현재 디렉터리 위치의 '절대경로' 조회
	- pwd

# 원하는 디렉터리로 이동
	- cd
	(1) cd .
	(2) cd ..		: 상위 디렉터리
	(3) cd ~ 		: 홈 디렉터리로 이동
	(4) cd /폴더명	: 특정 디렉터리로 이동

# 파일, 디렉터리 이동이나 이름변경
	-mv
	- 옵션
		(1) b	: 지워지는 것을 대비해 백업파일 생성 *
		(2) f	: 어떻게 처리할지 안보여줌
		(3) v	: 파일 옮기는 과정을 자세히 보여줌


# 안전히 종료
	(1)shutdown -h now	: 시스템 즉시 종료
	(2) shutdown -r now	: 시스템 즉시 재부팅

#  폴더 생성
	- mkdir
	- 옵션
		(1) p	: 만들려는 디렉터리의 상위 디렉터리도 만들어준다	*
		ex) mkdir -p aaa/bbb	: aaa폴더 아래에 bbb폴더까지 만들어준다

# 파일, 디렉터리 삭제
	- rm
	- 옵션
		(1) f : 안묻고 삭제
		(2) r : 파일은 그냥 지우고, 폴더면 하위것까지 전부 삭제 *
		(3) i : 사용자에게 물어보면서 삭제	
	* 권한이 있을경우에 가능
	* rm -rf 사용 금지 : os 모든 데이터 삭제 명령어 *

# 텍스트 파일 내용 출력하기
	- cat	* 폴더가 아닌 텍스트파일 내용을 출력하는 기능이다
	- 옵션
		(1) cat /etc/passwd > /파일명		: 기존의 파일(passwd)내용을 다른 파일명으로 덮어쓰기
		(2) cat /etc/passwd >> /파일명	: 기존의 파일에 내용을 추가

# 파일의 내용중 처음부터, 마지막부터 조회하기
	(1) head -숫자 파일명
	(2) tail - 숫자 파일명

# 복사할이름 
	- cp
	- 옵션
		(1) f	: 강제복사
		(2) r 	: 해당 디렉터리 복사할 경우 하위 디렉터리와 파일 모두 복사
		(3) p	: 원본파일의 소유주, 그룹, 권한. 시간정보를 보존하여 복사

# 현재 실행중인 프로세스 조회 / 권한 확인 / 셸 확인
	- ls
	- 옵션
		(1) r	: 내림차순정렬
		(2) t	: 파일이 수정된 시간을 기준으로 정렬
		(3) a	: 경로안 모든 .(숨긴파일)포함하여 출력
		(4) l	: 파일을 나열할 때 자세히 출력		
		(5) R	: 위치한 디렉터리 하부 디렉터리 파일까지 모두 출력
	- 사용문장
	(1) ls - al 		: 상세히 조회 
	ex) -rwxr -xr-x. 1 root root 95 dec 13 15:19 forsd.sh
	     '사용자, 그룹, other, 파일소유주, 부모프로세스, cpu, 날짜, 쉘' 을 의미
	(2) ls - rtl 		: 위의 세가지 옵션을 사용하여 정렬(숨김파일없이)
